Editor

The plan was to have a fully functional editor for the cart and the character, but it had to be cut because of placement of meshes.
Even though it was cut, I used the code of the editor to make two carts that you could choose between.

The way the editor works is that each part type has a list in a scriptable object which the user could go through and select the parts. When a part has been selected, the index of that part is saved in the scriptable object to be used later when loading in the vehicle. However, because I could not get it finished in time, we chose to create two vehicles that the users could pick.

Vehicle Loader

To load the vehicle, the code goes through all the lists and instantiates the parts at the selected index, giving them a appropriate name.
